# VSCode Lama2 README

Lama2 is a VSCode support extension for [Lama2](https://github.com/HexmosTech/Lama2).
## Features

1. Execute the focused `.l2` file through `Execute this file`
## Requirements

Run:

```
./install.sh
```

Restart VSCode
## Extension Settings

No settings presently.
